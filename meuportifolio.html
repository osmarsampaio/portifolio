Sobre

Café e código resume bem como Osmar trabalha: pouco discurso, muita entrega. Cada projeto no portfólio nasceu de um problema concreto — compartilhar arquivo sem depender de internet, digitalizar o anúncio de um outdoor, dar vida a um site de assistência técnica ou presentear alguém com uma página só dela.

O resultado é um perfil versátil: back-end em Python/Flask e Node.js, front-end em JavaScript, HTML e CSS, e conforto tanto para construir uma API quanto para deixar a interface bonita de usar.

JavaScript Python Node.js Flask Socket.io WebRTC HTML / CSS Docker

osmarsampaio/p2p-wifi JavaScript

WiFi Share — transferência P2P de arquivos

Envia qualquer arquivo entre dois dispositivos direto pelo navegador, sem instalar app e sem passar pela nuvem: a conexão é WebRTC ponto-a-ponto, com Socket.io só para o pareamento inicial via QR code. Testado com arquivos de até 4GB.

Node.js WebRTC Socket.io Docker
